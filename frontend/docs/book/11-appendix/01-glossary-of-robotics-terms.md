# Part 11: Appendix

## 01-Glossary of Robotics Terms

This glossary provides definitions for common terms encountered in the field of robotics. Understanding these terms is fundamental to comprehending the concepts and technologies discussed throughout this course book.

---

**A**

*   **Actuator:** A component that converts energy (electrical, hydraulic, pneumatic) into mechanical motion to perform a physical action. Examples include motors, solenoids, and pneumatic cylinders.
*   **Actuator, Hydraulic:** An actuator that uses pressurized incompressible fluid to generate powerful linear or rotary motion.
*   **Actuator, Pneumatic:** An actuator that uses compressed air to generate linear or rotary motion.
*   **ADC (Analog-to-Digital Converter):** An electronic circuit that converts a continuous analog voltage signal into a discrete digital value. Essential for microcontrollers to read analog sensors.
*   **AI (Artificial Intelligence):** The simulation of human intelligence in machines that are programmed to think like humans and mimic their actions, encompassing learning, problem-solving, and perception.
*   **Algorithm:** A finite, unambiguous set of instructions for solving a problem or performing a computation.
*   **Analog Signal:** A signal that varies continuously over time, taking on any value within a given range (e.g., a varying voltage from a potentiometer).
*   **Arduino:** An open-source electronics platform based on easy-to-use hardware and software, popular for prototyping and learning microcontrollers.
*   **Autonomous Robot:** A robot capable of operating and making decisions independently, without continuous human intervention, within a defined environment.
*   **Autonomy, Levels of:** A classification describing the degree of independence a robot has, ranging from human-operated to fully autonomous.
*   **AVR Microcontroller:** A family of microcontrollers developed by Atmel (now Microchip), known for their efficiency and used in many Arduino boards.

---

**B**

*   **Back-EMF (Back Electromotive Force):** A voltage generated by a motor as its coils cut through magnetic flux lines when it rotates. It opposes the applied voltage and is proportional to the motor's speed.
*   **Baud Rate:** The rate at which information is transferred in a serial communication channel, typically measured in bits per second (bps).
*   **Behavior Tree (BT):** A hierarchical, tree-like structure used to model complex robot behaviors. It offers modularity and reactivity, common in game AI and robotics.
*   **Bias (Neural Network):** An additional parameter in a neuron that allows the activation function to be shifted, effectively making the neuron more or less sensitive.
*   **Binary:** A number system with two symbols (0 and 1), used in digital electronics and computing.
*   **Bit:** A single binary digit (0 or 1).
*   **BLDC Motor (Brushless DC Motor):** An electric motor that uses electronic commutation instead of brushes, offering higher efficiency, longer lifespan, and more power.
*   **Blob (Computer Vision):** A region in an image characterized by uniform properties (e.g., color, intensity) that is distinct from its surroundings.
*   **Boolean Algebra:** A mathematical system for analyzing and simplifying digital circuits, using variables with two values (true/false or 0/1).
*   **Bounding Box (Computer Vision):** A rectangular box that encloses a detected object in an image, typically defined by its top-left corner (x,y) and its width and height (w,h).
*   **Breadboard:** A solderless construction base used for prototyping electronic circuits.
*   **Brownout:** A state where the power supply voltage drops below the minimum operating voltage of an electronic component, potentially causing erratic behavior or resets.
*   **Buck Converter:** A type of switching voltage regulator that steps down a DC input voltage to a lower DC output voltage.
*   **Byte:** A group of 8 bits.

---

**C**

*   **Calibration:** The process of adjusting sensor measurements or system parameters to improve accuracy and consistency, compensating for biases, non-linearities, and systematic errors.
*   **Camera, Monocular:** A standard 2D camera that captures a single view of a scene.
*   **Camera, RGB-D:** A camera that provides both a standard 2D color image (RGB) and a per-pixel depth map (D).
*   **Camera, Stereo:** A system using two (or more) cameras placed a known distance apart, mimicking human binocular vision, to calculate depth information via triangulation.
*   **Capacitor:** An electronic component that stores electrical energy in an electric field.
*   **Centroid:** The geometric center of a shape or object (e.g., a detected contour in computer vision).
*   **Chassis:** The structural frame or base of a robot that houses its components.
*   **Closed-Loop Control:** A control system that uses feedback from the system's output to adjust the input, continuously working to reduce the error between desired and actual states.
*   **Clustering (Unsupervised Learning):** The task of grouping a set of objects into subsets (clusters) such that objects in the same cluster are more similar to each other than to those in other clusters.
*   **CNN (Convolutional Neural Network):** A type of deep learning neural network specifically designed for processing grid-like data, particularly images, using convolutional layers to learn spatial hierarchies of features.
*   **Cobot (Collaborative Robot):** A robot designed to work safely and interactively alongside humans in a shared workspace.
*   **Color Space:** A mathematical model that describes how colors can be represented as numbers (e.g., RGB, HSV).
*   **Commutation:** The process of switching the direction of current in a motor's windings to maintain continuous rotation.
*   **Complementary Filter:** A simple sensor fusion algorithm used to combine two sensor readings with complementary noise characteristics (e.g., accelerometer and gyroscope for stable orientation).
*   **Computer Vision (CV):** An interdisciplinary field that deals with how computers can gain high-level understanding from digital images or videos.
*   **Control Flow:** The order in which instructions are executed in a program (e.g., sequential, conditional, looped).
*   **Controller:** The "brain" of a robot system (e.g., microcontroller, computer) that processes sensor data, executes programs, and sends commands to actuators.
*   **Contour (Computer Vision):** A curve joining all continuous points along the boundary of an object in an image.
*   **Coordinate System:** A reference framework used to define the position and orientation of objects in space.
*   **CPU (Central Processing Unit):** The "brain" of a computer or microcontroller, responsible for executing instructions and performing calculations.
*   **Cross-Entropy Loss:** A common loss function used in classification tasks, measuring the difference between predicted probability distributions and true distributions.
*   **Cross-Talk (Ultrasonic):** Interference between multiple ultrasonic sensors where one sensor's echo is mistakenly picked up by another.

---

**D**

*   **DAC (Digital-to-Analog Converter):** An electronic circuit that converts a digital value into a continuous analog voltage signal.
*   **Data Structure:** A particular way of organizing and storing data in a computer so that it can be accessed and modified efficiently.
*   **Deep Learning (DL):** A subset of machine learning that uses neural networks with a large number of hidden layers to automatically learn hierarchical features from data.
*   **Deep Reinforcement Learning (DRL):** The combination of deep learning with reinforcement learning, where neural networks act as policy or value function approximators.
*   **Deformable Object:** An object whose shape can change under external forces (e.g., cloth, ropes).
*   **Denavit-Hartenberg (DH) Convention:** A standardized method for defining coordinate frames for each joint in a serial kinematic chain, simplifying the derivation of forward kinematics.
*   **Derivative (PID):** The term in a PID controller that responds to the rate of change of the error, providing damping and predicting future error.
*   **Descriptor (Feature):** A compact, invariant numerical representation of the local image patch around a detected feature (e.g., SIFT, ORB).
*   **Digital Signal:** A signal that represents information using discrete values, typically two states (binary 0 and 1).
*   **Digital Twin:** A virtual replica of a physical robot or system, used for simulation, monitoring, and testing.
*   **Dimensionality Reduction (Unsupervised Learning):** The process of reducing the number of variables in a dataset while retaining most of the important information.
*   **Diode:** An electronic component that allows current to flow primarily in one direction.
*   **Disparity (Stereo Vision):** The difference in the horizontal positions of corresponding points in two images captured by a stereo camera system. Inversely proportional to depth.
*   **DOF (Degrees of Freedom):** The number of independent parameters that define the configuration of a mechanical system (e.g., 6 DOF for a robot arm in 3D space).
*   **DRL (Deep Reinforcement Learning):** See Deep Reinforcement Learning.
*   **Drift (IMU, Odometry):** The accumulation of small errors over time in state estimation, leading to a deviation from the true position or orientation.
*   **Duty Cycle (PWM):** The ratio of the "on" time to the total period of a pulse width modulated (PWM) signal, typically expressed as a percentage.

---

**E**

*   **Echo Pin (Ultrasonic):** The pin on an ultrasonic sensor module that receives the reflected sound wave.
*   **Edge Detection (Computer Vision):** Algorithms that identify points in an image where the image brightness changes sharply, typically marking the boundaries of objects.
*   **EEPROM (Electrically Erasable Programmable Read-Only Memory):** A type of non-volatile memory in microcontrollers used to store small amounts of data that need to persist when power is off.
*   **Encoder:** An electromechanical device that converts angular or linear position into an electrical signal, typically digital pulses, used for measuring position and speed.
*   **Encoder, Absolute:** An encoder that provides a unique digital code for each position, knowing its absolute position even after power cycling.
*   **Encoder, Incremental:** An encoder that generates a series of pulses as it moves, requiring a controller to count pulses to track relative position and direction.
*   **End-Effector:** The "hand" or tool at the end of a robot arm, designed to interact with the environment (e.g., grippers, welders).
*   **Environment (RL):** The world in which a Reinforcement Learning agent operates, providing states, rewards, and responding to actions.
*   **ESP32/ESP8266:** Series of low-cost, low-power microcontrollers with integrated Wi-Fi and (for ESP32) Bluetooth, popular for IoT and connected robotics.
*   **Euclidean Distance:** The straight-line distance between two points in Euclidean space, often used as a distance metric in algorithms like K-Means.
*   **Exploration/Exploitation (RL):** The dilemma in Reinforcement Learning where an agent must choose between trying new actions (exploration) to find better rewards or sticking to known good actions (exploitation).
*   **Exteroceptive Sensor:** A sensor that measures information about the robot's external environment (e.g., camera, LiDAR, ultrasonic).
*   **External Interrupt:** An interrupt triggered by a change in the logic level on a specific digital pin of a microcontroller.

---

**F**

*   **Feature (Computer Vision):** A distinctive, robust, and informative point, line, or region in an image used for tasks like object recognition or tracking.
*   **Feature Detection:** The process of identifying "interesting" points or regions in an image.
*   **Feature Description:** Creating a compact, invariant numerical representation of the local image patch around a detected feature.
*   **Feature Matching:** Finding corresponding features between different images or across consecutive frames of a video.
*   **Feedback Control:** See Closed-Loop Control.
*   **FIFO (First-In, First-Out):** A data structure where the first element added is the first one to be removed (e.g., a queue).
*   **Finite State Machine (FSM):** A mathematical model of computation that can be in exactly one of a finite number of states at any given time, with transitions between states triggered by events.
*   **Flash Memory:** A type of non-volatile memory in microcontrollers used to store the program code and constant data.
*   **Forward Kinematics (FK):** The process of calculating the position and orientation of a robot's end-effector given its joint angles and link lengths.
*   **FSR (Force Sensitive Resistor):** A passive electronic component whose electrical resistance changes when pressure is applied to its surface.
*   **Full-Duplex:** A communication system that allows simultaneous two-way data transmission (e.g., some UART implementations, SPI).

---

**G**

*   **GAN (Generative Adversarial Network):** A type of deep learning model consisting of a generator and a discriminator, used for generating new data samples that resemble training data.
*   **GND (Ground):** The common reference point in an electrical circuit, typically 0V.
*   **GPIO (General Purpose Input/Output):** Configurable pins on a microcontroller that can be set as either inputs (to read signals) or outputs (to send signals).
*   **Grasping:** The act of picking up and holding an object with a robot gripper or end-effector.
*   **Gripper:** A type of end-effector designed to grasp and manipulate objects.
*   **Gyroscopes:** Sensors that measure angular velocity (rate of rotation) around an axis.

---

**H**

*   **Half-Duplex:** A communication system that allows two-way data transmission, but only in one direction at a time (e.g., I2C).
*   **Hall Effect Sensor:** A sensor that detects the presence and strength of a magnetic field.
*   **Hamming Distance:** A metric used to compare binary feature descriptors, counting the number of positions at which the corresponding bits are different.
*   **Hand-crafted Features:** Features extracted from data (e.g., images) using explicit algorithms designed by humans, as opposed to learned features.
*   **Hard-Iron Distortion:** Magnetic interference caused by permanently magnetized materials near a magnetometer.
*   **H-Bridge:** An electronic circuit configuration that allows a DC motor to be driven in both forward and reverse directions and to be braked.
*   **HC-SR04:** A common, low-cost ultrasonic distance sensor module.
*   **Heuristic Function (A*):** An estimated cost function from the current node to the goal node in search algorithms like A*, used to guide the search.
*   **HSV (Hue, Saturation, Value):** A color space that is often preferred for color-based object detection because its Hue component is relatively stable under varying illumination.
*   **Human-Robot Interaction (HRI):** The study of interactions between humans and robots, focusing on design, communication, and social implications.

---

**I**

*   **I2C (Inter-Integrated Circuit):** A synchronous, two-wire (SDA, SCL), multi-master, multi-slave serial communication protocol commonly used for connecting low-speed peripheral ICs.
*   **IDE (Integrated Development Environment):** A software application that provides comprehensive facilities to computer programmers for software development (e.g., Arduino IDE).
*   **IMU (Inertial Measurement Unit):** An electronic device that measures and reports a body's specific force (acceleration), angular rate (rotation), and sometimes magnetic field, using accelerometers, gyroscopes, and magnetometers.
*   **Inductor:** An electronic component that stores energy in a magnetic field when electric current flows through it.
*   **Inference (Deep Learning):** The process of using a trained machine learning model to make predictions or decisions on new, unseen data.
*   **Inheritance (OOP):** A mechanism where a new class (derived class) can inherit properties and behaviors from an existing class (base class), promoting code reuse.
*   **Input Layer (Neural Network):** The first layer of a neural network that receives the raw input data.
*   **Instance Segmentation:** A type of image segmentation that identifies and delineates each individual object instance in the image, distinguishing between multiple objects of the same class.
*   **Integral (PID):** The term in a PID controller that accumulates past errors over time to eliminate steady-state error.
*   **Integral Windup:** A situation in PID controllers where a large error causes the integral term to accumulate excessively, leading to overshoot and instability.
*   **Integrated Circuit (IC):** A miniature electronic circuit fabricated on a single piece of semiconductor material (microchip).
*   **Interfacing:** The process of connecting and enabling communication between different hardware or software components.
*   **Interrupt:** A mechanism that allows a microcontroller to respond immediately to external or internal events, temporarily suspending its current task to execute an Interrupt Service Routine (ISR).
*   **Interrupt Service Routine (ISR):** A special function executed when an interrupt occurs.
*   **Inverse Kinematics (IK):** The process of calculating the required joint angles or displacements of a robot manipulator to achieve a desired position and orientation of its end-effector.
*   **IoU (Intersection over Union):** A metric used to evaluate the accuracy of a predicted bounding box or segmentation mask against a ground truth.
*   **IR Sensor (Infrared Sensor):** A sensor that uses infrared light to detect objects, measure distance, or follow lines.
*   **Isolation Forest:** An unsupervised machine learning algorithm particularly effective for anomaly detection by isolating outliers.

---

**J**

*   **Jacobian Matrix:** A matrix of all first-order partial derivatives of a vector-valued function. Used in Extended Kalman Filters for linearization and in inverse kinematics.
*   **JTAG (Joint Test Action Group):** An industry standard for debugging embedded systems, allowing for hardware-level debugging.
*   **Jumper Wires:** Wires with connectors (male/female) used to make temporary connections on breadboards or between components.

---

**K**

*   **Kalman Filter (KF):** An optimal recursive data processing algorithm that combines measurements with a prediction of a system's state to produce an estimate that is statistically more accurate. Assumes linear systems and Gaussian noise.
*   **K-Means Clustering:** An iterative unsupervised learning algorithm that partitions data points into 'k' distinct clusters based on similarity.
*   **Kinematics:** The study of motion without considering the forces that cause it.
*   **K-NN (K-Nearest Neighbors):** A simple supervised learning algorithm that classifies a new data point by finding the 'k' nearest data points in the training set and assigning the most common class label.
*   **Kp, Ki, Kd (PID Gains):** The proportional, integral, and derivative gain constants in a PID controller that determine the aggressiveness of each term's contribution to the control output.

---

**L**

*   **L298N:** A common, low-cost dual H-bridge motor driver IC, often available as a module, used for controlling DC motors.
*   **LDR (Light Dependent Resistor):** See Photoresistor.
*   **LED (Light Emitting Diode):** A semiconductor light source.
*   **LIFO (Last-In, First-Out):** A data structure where the last element added is the first one to be removed (e.g., a stack).
*   **LiDAR (Light Detection and Ranging):** A remote sensing method that uses pulsed laser light to measure distances to the Earth or objects, creating 2D or 3D maps.
*   **Limit Switch:** A mechanical switch that is actuated by contact with a moving part to detect the end of travel or presence of an object.
*   **Linear Regression:** A supervised learning algorithm that models the linear relationship between a dependent variable and one or more independent variables, often used for prediction or calibration.
*   **LiPo Battery (Lithium Polymer):** A type of rechargeable battery known for high energy density and high discharge rates, popular in drones and portable electronics.
*   **Localization:** The process by which a robot determines its position and orientation within an environment.
*   **Logic Gate:** A fundamental electronic component that performs a logical operation on one or more binary inputs to produce a single binary output (e.g., AND, OR, NOT).
*   **Logic Analyzer:** A hardware tool used to capture and display digital signals over time, useful for debugging digital communication protocols.
*   **Logic Levels:** The distinct voltage ranges used to represent binary 0 (LOW) and 1 (HIGH) in digital electronics.
*   **Loop Closure (SLAM):** The process in SLAM where the robot recognizes that it has returned to a previously visited location, which is crucial for correcting accumulated drift and optimizing the map.
*   **Loss Function (Deep Learning):** A mathematical function that quantifies the difference between the predicted output of a model and the true target value, guiding the learning process.

---

**M**

*   **Magnetometer:** A sensor that measures the strength and direction of magnetic fields, often used as an electronic compass to determine absolute heading.
*   **Map, Occupancy Grid:** A grid-based representation of an environment where each cell indicates the probability of being occupied by an obstacle.
*   **Map, Semantic:** A map that not only includes geometric information but also labels areas or objects with semantic meaning (e.g., "road," "wall," "person").
*   **Mapping:** The process by which a robot builds a representation (map) of its environment.
*   **MCU (Microcontroller Unit):** See Microcontroller.
*   **Mean Squared Error (MSE):** A common loss function and evaluation metric in regression, measuring the average of the squares of the errors.
*   **Median Filter:** A non-linear digital filtering technique, often used for noise reduction, that replaces each pixel or sample value with the median of its neighbors.
*   **MEMS (Micro-Electro-Mechanical Systems):** Microscopic devices that combine electrical and mechanical components, often used in sensors like accelerometers and gyroscopes.
*   **Memory Fragmentation:** A phenomenon in dynamic memory allocation where free memory is broken into many small, non-contiguous blocks, making it difficult to allocate larger blocks.
*   **Memory Leak:** A type of resource leak that occurs when a computer program incorrectly manages memory allocations, leading to a build-up of unused memory that is not returned to the operating system.
*   **Merge Sort:** A divide-and-conquer sorting algorithm that guarantees O(N log N) time complexity in all cases.
*   **ML (Machine Learning):** A subset of AI that focuses on building systems that can learn from data without being explicitly programmed.
*   **MLP (Multi-Layer Perceptron):** A type of feedforward neural network, the simplest form of deep learning.
*   **Modular Design:** A design principle that involves dividing a system into smaller, independent, and interchangeable components (modules).
*   **Morphological Operations (Computer Vision):** Image processing operations that process images based on shapes, typically applied to binary images (e.g., erosion, dilation).
*   **MOSFET (Metal-Oxide-Semiconductor Field-Effect Transistor):** A type of transistor commonly used as an electronic switch, particularly for controlling higher currents.
*   **Motor Driver:** An electronic circuit that acts as an interface between a microcontroller and a motor, providing the necessary current and voltage to drive the motor.
*   **Motor, DC (Direct Current):** An electric motor that converts DC electrical energy into mechanical energy, typically rotation.
*   **Motor, Stepper:** An electric motor that divides a full rotation into a number of equal steps, allowing for precise position control without a feedback sensor.
*   **Motor, Servo (Hobby):** A geared DC motor with an integrated position sensor and control circuit, allowing it to hold a specific angular position.
*   **MPU6050:** A popular 6-DOF (Degrees of Freedom) IMU sensor module containing a 3-axis accelerometer and a 3-axis gyroscope.
*   **MSE (Mean Squared Error):** See Mean Squared Error.
*   **Multi-Master (I2C):** A capability in I2C where multiple devices can act as the master controller on the same bus.
*   **Multi-Slave (I2C, SPI):** A capability where a single master can communicate with multiple slave devices on the same bus.
*   **MVP (Minimum Viable Product):** The smallest possible set of features that makes a product functional and usable, often a target for hackathons.

---

**N**

*   **NAND Gate:** A logic gate whose output is LOW only if all inputs are HIGH; otherwise, the output is HIGH. (NOT AND).
*   **Natural Language Processing (NLP):** A field of AI that deals with the interaction between computers and human language, enabling computers to understand, interpret, and generate human speech and text.
*   **Neural Network (NN):** A computational model inspired by the structure of biological neural networks, consisting of interconnected neurons organized in layers, used for learning patterns from data.
*   **NiMH Battery (Nickel-Metal Hydride):** A type of rechargeable battery.
*   **NLU (Natural Language Understanding):** The part of NLP that focuses on comprehending the meaning and intent behind human language.
*   **Noise (Sensor):** Random fluctuations or inaccuracies in sensor readings that deviate from the true value.
*   **Non-Blocking Code:** Programming techniques (e.g., using `millis()` instead of `delay()`) that allow a microcontroller to perform multiple tasks concurrently without pausing execution.
*   **Non-Maximum Suppression (NMS):** An algorithm used in object detection to filter out redundant bounding boxes, keeping only the most confident one among overlapping detections.
*   **NOR Gate:** A logic gate whose output is HIGH only if all inputs are LOW; otherwise, the output is LOW. (NOT OR).
*   **Normal Distribution (Gaussian):** A common probability distribution that is symmetric around its mean, often used to model sensor noise.
*   **NPN Transistor:** A type of bipolar junction transistor.
*   **NTC (Negative Temperature Coefficient):** A thermistor whose resistance decreases as temperature increases.

---

**O**

*   **Object (OOP):** An instance of a class, a concrete entity created based on the class blueprint.
*   **Object Detection (Computer Vision):** Identifying the presence and location (bounding box) of specific objects in an image or video.
*   **Object Recognition (Computer Vision):** Identifying *what* a detected object is, assigning a class label.
*   **Odometry:** The use of data from motion sensors (primarily encoders) to estimate a robot's change in position and orientation over time.
*   **Ohm's Law:** A fundamental law in electronics stating that the current flowing through a conductor between two points is directly proportional to the voltage across the two points and inversely proportional to the resistance between them (V = I * R).
*   **OpenCV (Open Source Computer Vision Library):** A comprehensive, open-source library for computer vision, machine learning, and image processing.
*   **Open-Loop Control:** A control system where the control action is independent of the system's output (no feedback).
*   **Operating System (OS):** System software that manages computer hardware and software resources and provides common services for computer programs.
*   **Optimal Path:** The shortest or most efficient path between two points in a graph, often considering edge weights (costs).
*   **Optimization (Deep Learning):** The process of adjusting the weights and biases of a neural network during training to minimize the loss function.
*   **OR Gate:** A logic gate whose output is HIGH if any input is HIGH.
*   **ORB (Oriented FAST and Rotated BRIEF):** A fast and robust feature detector and descriptor, often used as a free alternative to SIFT/SURF.
*   **Oscilloscope:** An electronic test instrument that graphically displays varying signal voltages, useful for debugging analog circuits and PWM signals.
*   **Otsu's Binarization:** An automatic method for determining the optimal global threshold value for image binarization.
*   **Output Layer (Neural Network):** The final layer of a neural network that produces the prediction or result.
*   **Overfitting:** A phenomenon in machine learning where a model performs very well on the training data but poorly on new, unseen data, typically due to learning noise or irrelevant features.

---

**P**

*   **Panoptic Segmentation:** A type of image segmentation that combines semantic and instance segmentation, assigning a class label to every pixel and distinguishing individual object instances for "things."
*   **Parallel Circuit:** An electric circuit where components are connected across the same two points, creating multiple paths for current to flow, with the same voltage across each component.
*   **Particle Filter (PF):** A non-parametric, non-linear filter that represents the probability distribution of a system's state using a set of weighted random samples (particles), commonly used for robot localization.
*   **Path Planning:** The process of finding a sequence of movements (path) for a robot from a start point to a goal point while avoiding obstacles.
*   **PCA (Principal Component Analysis):** A linear dimensionality reduction technique that identifies the directions (principal components) of greatest variance in a dataset.
*   **PCB (Printed Circuit Board):** A board that mechanically supports and electrically connects electronic components using conductive tracks, pads, and other features etched from copper sheets.
*   **PCL (Point Cloud Library):** An open-source library for processing 3D point cloud data.
*   **Perception:** The process by which a robot acquires and interprets information from its environment through various sensors.
*   **Peripheral:** An external device that connects to a computer or microcontroller (e.g., sensor, motor, display).
*   **Photodiode:** A semiconductor device that converts light into electrical current.
*   **Photoresistor (LDR):** A passive electronic component whose electrical resistance decreases as the intensity of light falling on it increases.
*   **Phototransistor:** A transistor whose current flow is controlled by light.
*   **PID (Proportional-Integral-Derivative) Control:** A common feedback control algorithm that calculates an error value as the difference between a measured process variable and a desired setpoint, and applies a correction based on proportional, integral, and derivative terms.
*   **Pinhole Camera Model:** A mathematical model that describes the geometric relationship between a 3D point in the world and its projection onto a 2D image plane.
*   **Pixel:** The smallest unit of a digital image, arranged in a grid.
*   **Plant (Control System):** The system or process that is being controlled (e.g., a motor, a robot arm).
*   **PLC (Programmable Logic Controller):** An industrial digital computer that has been ruggedized and adapted for the control of manufacturing processes.
*   **PNP Transistor:** A type of bipolar junction transistor.
*   **Point Cloud:** A collection of 3D data points, where each point represents a single measurement from a 3D sensor (e.g., LiDAR, RGB-D camera).
*   **Policy (RL):** A strategy in Reinforcement Learning that maps states to actions, dictating the agent's behavior.
*   **Policy Gradient:** A class of Reinforcement Learning algorithms that directly learn a parameterized policy function to maximize expected reward.
*   **Polymorphism (OOP):** The ability for objects of different classes to be treated as objects of a common base class, allowing different implementations of a method to be called through a single interface.
*   **Potentiometer:** A three-terminal resistor with a rotating contact that acts as a variable voltage divider, often used to measure angular position or provide variable input.
*   **PPR (Pulses Per Revolution):** A specification for encoders, indicating the number of electrical pulses generated for each full rotation of the encoder shaft.
*   **Preprocessing (Image):** Operations performed on an image to improve its quality, extract useful information, or prepare it for further analysis (e.g., noise reduction, contrast adjustment).
*   **Principal Components (PCA):** The new set of orthogonal (uncorrelated) features identified by PCA that capture the most variance in the data.
*   **Proportional (PID):** The term in a PID controller that generates control output proportional to the current error.
*   **Proximity Sensor:** A sensor that detects the presence of nearby objects without physical contact.
*   **PROGMEM (Arduino):** A keyword in Arduino (AVR-specific) used to store constant data (especially large arrays or strings) in Flash memory instead of SRAM, saving precious RAM.
*   **Proprioceptive Sensor:** A sensor that measures the robot's internal state (e.g., joint angles, wheel rotations, battery level).
*   **PPM (Parts Per Million):** A unit of concentration, often used for gas sensors.
*   **PWM (Pulse Width Modulation):** A technique for controlling the average power delivered to a device by varying the "on" time (duty cycle) of a pulse.
*   **PySerial:** A Python library that provides serial port access, allowing Python programs to communicate with microcontrollers or other serial devices.
*   **Python:** A high-level, interpreted programming language widely used in robotics for high-level control, AI/ML, and rapid prototyping.

---

**Q**

*   **Q-Learning:** A model-free, off-policy Reinforcement Learning algorithm that learns the optimal Q-value function for a given environment.
*   **Q-Table (Q-Learning):** A table storing Q-values for all possible state-action pairs in an environment.
*   **Quadrature Encoding:** A method used by incremental encoders that provides two output signals (A and B) 90 degrees out of phase, allowing the controller to determine both direction and higher resolution.
*   **Queue (FIFO):** A data structure where elements are added to one end (rear) and removed from the other end (front) following a "First-In, First-Out" principle.
*   **Quick Sort:** A highly efficient, comparison-based sorting algorithm with average O(N log N) time complexity.

---

**R**

*   **RAM (Random-Access Memory):** Volatile memory used for temporary data storage during program execution.
*   **Raspberry Pi:** A series of small, single-board computers, often used in robotics for more complex tasks requiring an operating system, more processing power, and camera capabilities.
*   **Reactive Decision Making:** Simple "if-then-else" rules that trigger immediate actions based on current sensor readings, without much foresight.
*   **ReLU (Rectified Linear Unit):** A common activation function used in neural networks, outputting the input directly if positive, and zero otherwise.
*   **Regression (Supervised Learning):** A supervised learning task where the model predicts a continuous numerical value for a given input.
*   **Reinforcement Learning (RL):** A paradigm in Machine Learning where an agent learns to make a sequence of decisions in an environment to maximize a cumulative reward through trial and error.
*   **Reprojection Error:** In visual SLAM, the difference between a 3D feature point projected onto an image and its observed 2D location, used as an optimization metric.
*   **Resistor:** An electronic component that opposes the flow of electric current.
*   **Resolution (Image):** The total number of pixels (width x height) in an image.
*   **Reward (RL):** A numerical feedback signal from the environment indicating the desirability of an action taken by an agent in a particular state.
*   **RGB (Red, Green, Blue):** A color space model where colors are represented by combining varying intensities of red, green, and blue light.
*   **RGB-D Camera:** See Camera, RGB-D.
*   **RND (Resistance Temperature Detector):** A highly accurate temperature sensor whose resistance changes linearly with temperature.
*   **Robot Operating System (ROS):** A flexible framework for writing robot software, providing middleware, tools, and libraries.
*   **Rotor:** The rotating part of an electric motor.
*   **RPM (Revolutions Per Minute):** A unit of rotational speed.
*   **RNN (Recurrent Neural Network):** A type of neural network designed for processing sequential data, with internal memory.
*   **RUL (Remaining Useful Life):** A prediction of how much longer a component or system will function before failure.

**S**

*   **SDA (Serial Data Line):** The data line in I2C communication.
*   **SCL (Serial Clock Line):** The clock line in I2C communication.
*   **SDK (Software Development Kit):** A set of software development tools that allows the creation of applications for a certain software package, hardware platform, or game console.
*   **Segmentation, Image:** The process of partitioning an image into multiple segments (sets of pixels), typically to identify objects or regions with similar properties.
*   **Segmentation, Instance:** Image segmentation that identifies and delineates each individual object instance in an image.
*   **Segmentation, Panoptic:** Image segmentation that combines semantic and instance segmentation.
*   **Segmentation, Semantic:** Image segmentation that classifies every pixel in an image into a predefined category.
*   **Selection Sort:** A simple sorting algorithm that repeatedly finds the minimum element from the unsorted part of a list and puts it at the beginning of the sorted part.
*   **Sensor:** A device that detects and responds to some type of input from the physical environment (e.g., light, heat, motion) and converts it into an electrical signal.
*   **Sensor Fusion:** The process of combining data from multiple sensors to achieve a more accurate, reliable, and complete estimate of a robot's state and its environment.
*   **Sequence (Behavior Tree):** A control flow node in a behavior tree that executes its children in order until one fails, then returns FAILURE. If all succeed, returns SUCCESS.
*   **Serial Communication:** A method of data transmission where data bits are sent sequentially over a single channel.
*   **Serial Monitor (Arduino):** A feature in the Arduino IDE that allows for sending and receiving text data between the Arduino board and the computer, useful for debugging.
*   **Serial Plotter (Arduino):** A feature in the Arduino IDE that visualizes numerical data from the Arduino in real-time graphs.
*   **Series Circuit:** An electric circuit where components are connected end-to-end, forming a single path for current to flow, with the same current through each component.
*   **Servo Motor (Hobby):** See Motor, Servo (Hobby).
*   **Set Point (Control System):** The desired output or target state for a controlled system.
*   **SFM (Structure from Motion):** A photogrammetric range imaging technique for estimating three-dimensional structures from a sequence of 2D images.
*   **Sigmoid Function:** A common activation function in neural networks that squashes input values into a range between 0 and 1.
*   **SIFT (Scale-Invariant Feature Transform):** A highly distinctive and robust feature descriptor, invariant to scale, rotation, and illumination changes.
*   **SLAM (Simultaneous Localization and Mapping):** The computational problem of building a map of an unknown environment while simultaneously keeping track of a robot's location within that map.
*   **Slave (Serial Communication):** A device on a serial bus that responds to commands from a master device.
*   **SMD (Surface Mount Device):** An electronic component mounted directly onto the surface of a PCB.
*   **Soft Robotics:** A subfield of robotics that deals with robots constructed from highly compliant materials, often mimicking biological structures.
*   **Soft-Iron Distortion:** Magnetic interference caused by magnetically permeable materials near a magnetometer.
*   **Softmax Function:** An activation function typically used in the output layer of neural networks for multi-class classification, converting a vector of numbers into a probability distribution.
*   **Solvent Trap:** A device attached to the barrel of a firearm to catch cleaning solvents. (Incorrect context, likely a typo or misinterpretation).
*   **Space Complexity:** A measure of how the memory usage of an algorithm grows with the size of its input.
*   **SPI (Serial Peripheral Interface):** A synchronous, four-wire, master-slave serial communication protocol generally faster than I2C, used for connecting higher-speed peripherals.
*   **SRAM (Static Random-Access Memory):** Volatile memory in microcontrollers used for temporary data storage (variables, stack, heap).
*   **SSD (Single Shot Detector):** A one-stage deep learning object detection algorithm known for its speed.
*   **Stack (LIFO):** A data structure where elements are added and removed from the same end, following a "Last-In, First-Out" principle.
*   **Stack Overflow:** An error that occurs when a program attempts to use more memory on the call stack than has been allocated for it, typically due to excessive recursion or large local variables.
*   **State (RL):** A complete description of the environment in a Reinforcement Learning problem.
*   **State Space Control:** A control system design approach that models the entire system using a set of internal variables (state variables) and matrices.
*   **State Variables:** A minimal set of variables that completely describe the state of a system at any given time.
*   **State Machine:** See Finite State Machine (FSM).
*   **Stator:** The stationary part of an electric motor.
*   **Steady-State Error:** A persistent difference between the desired setpoint and the actual output of a control system after transient responses have subsided.
*   **Stepper Motor:** See Motor, Stepper.
*   **Stereo Vision:** See Camera, Stereo.
*   **Structured Light (Depth Sensing):** A technique where a known pattern of light is projected onto a scene, and its deformation is captured by a camera to determine 3D shape and depth.
*   **Supervised Learning:** A Machine Learning paradigm where a model learns from labeled examples (input features and their corresponding correct output labels) to make predictions on new, unseen data.
*   **SURF (Speeded Up Robust Features):** A faster approximation of the SIFT feature descriptor.
*   **SWD (Serial Wire Debug):** A 2-pin interface used for debugging embedded systems, part of the ARM debug interface specification.

**T**

*   **Tactile Sensor:** A sensor that detects physical contact, pressure, or force, often providing a robot with a sense of touch.
*   **Tachometer:** A device used to measure the rotational speed of a shaft or disk.
*   **t-SNE (t-Distributed Stochastic Neighbor Embedding):** A non-linear dimensionality reduction technique primarily used for visualizing high-dimensional datasets.
*   **Teleoperation:** Remote control of a robot by a human operator.
*   **TensorFlow:** An open-source machine learning framework developed by Google, widely used for deep learning.
*   **Test Set:** A separate dataset used for the final, unbiased evaluation of a machine learning model's performance on truly unseen data.
*   **Thermistor:** A temperature-sensitive resistor whose resistance changes significantly with temperature.
*   **Thermocouple:** A temperature sensor consisting of two dissimilar metal wires joined at one end, which generates a voltage proportional to temperature.
*   **Thresholding (Image):** A segmentation technique that converts a grayscale image into a binary (black and white) image based on a pixel intensity threshold.
*   **Time Complexity:** A measure of how the running time of an algorithm grows with the size of its input.
*   **Timer (Microcontroller):** A hardware peripheral on a microcontroller used for precise timing, counting events, or generating PWM signals.
*   **Time-of-Flight (ToF) Sensor:** A sensor that measures distance by emitting a signal (light or sound) and measuring the time it takes for the signal to return after reflection.
*   **ToF Camera:** A camera that captures a depth map by measuring the time-of-flight for light to return for each pixel.
*   **Transducer:** A device that converts one form of energy into another (e.g., a sensor converting physical input to electrical signal).
*   **Transfer Learning:** A technique in machine learning where a model pre-trained on a large, general dataset is fine-tuned on a smaller, specific dataset for a related task.
*   **Transformation Matrix (Homogeneous):** A mathematical matrix that combines both rotation and translation into a single matrix, used to represent spatial relationships between coordinate frames.
*   **Transition (FSM):** The change from one state to another in a Finite State Machine, triggered by an event or condition.
*   **Transistor:** A semiconductor device that can amplify or switch electronic signals and electrical power.
*   **Trig Pin (Ultrasonic):** The pin on an ultrasonic sensor module that transmits the sound wave.
*   **Triangulation (Stereo Vision):** A geometric method used to calculate the 3D position of a point by finding its projection in two or more images taken from different viewpoints.
*   **Truth Table:** A mathematical table used in logic to compute the functional values of logical expressions on all combinations of their inputs.
*   **Tuning (PID):** The process of adjusting the Kp, Ki, Kd gains of a PID controller to achieve desired performance (e.g., stability, speed, accuracy).
*   **Two-Stage Detector (Object Detection):** Deep learning object detection models that first propose regions of interest and then classify and refine them.

**U**

*   **UKF (Unscented Kalman Filter):** An extension of the Kalman filter for non-linear systems that avoids linearization by using a deterministic sampling technique (sigma points).
*   **Ultrasonic Sensor:** A sensor that uses high-frequency sound waves to detect objects and measure distance.
*   **Unsupervised Learning:** A Machine Learning paradigm that deals with unlabeled data, aiming to discover hidden patterns, structures, or relationships within the data without explicit output labels.
*   **UART (Universal Asynchronous Receiver-Transmitter):** A common asynchronous serial communication protocol that uses two dedicated wires (TX, RX) for point-to-point data transmission.

---

**V**

*   **Validation Set:** A separate dataset used during machine learning model training to tune hyperparameters and prevent overfitting.
*   **Value Function (RL):** A prediction of future rewards in Reinforcement Learning, representing the expected total reward from a given state or state-action pair.
*   **Variable:** A named storage location that holds a value, which can change during program execution.
*   **Velocity, Angular:** The rate of change of angular position.
*   **Velocity, Linear:** The rate of change of linear position.
*   **Visual Odometry (VO):** The process of estimating the egomotion (self-motion) of a robot by analyzing a sequence of camera images.
*   **Visual Servoing:** Guiding a robot manipulator using visual feedback from a camera.
*   **Volatile (Keyword):** A keyword in C/C++ that tells the compiler that a variable's value can be changed by something outside the normal flow of the program (e.g., by an Interrupt Service Routine), preventing aggressive compiler optimizations.
*   **Voltage:** The electric potential difference or "pressure" that drives electric current, measured in Volts (V).
*   **Voltage Divider:** A simple series circuit used to create an output voltage that is a fraction of the input voltage.

---

**W**

*   **Watchdog Timer (WDT):** A hardware timer in microcontrollers that, if not reset periodically by the running program, will automatically reset the microcontroller, preventing it from freezing indefinitely.
*   **Weights (Neural Network):** Parameters in a neural network that determine the strength of connections between neurons, adjusted during training.
*   **Weighted Sum:** The sum of inputs multiplied by their respective weights, calculated by a neuron.
*   **Wi-Fi:** A wireless networking technology that allows devices to connect to a local area network (LAN) and the Internet.
*   **Wire.h (Arduino):** The Arduino library for I2C communication.
*   **Wireless Communication:** Data transmission without physical cables (e.g., Wi-Fi, Bluetooth, radio).
*   **World Frame:** A fixed reference coordinate system defining the robot's environment.

---

**X**

*   **XOR Gate:** A logic gate whose output is HIGH if the inputs are different; otherwise, the output is LOW. (Exclusive OR).

---

**Y**

*   **YOLO (You Only Look Once):** A one-stage deep learning object detection algorithm known for its speed, processing entire images to predict bounding boxes and class probabilities in a single pass.

---

**Z**

*   **Ziegler-Nichols Method:** A systematic approach for tuning PID controller gains.
